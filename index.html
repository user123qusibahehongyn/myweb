领养须知

领养条件

  • 年满18周岁，具有完全民事行为能力
  • 有固定的住所和稳定的收入
  • 能够提供适合宠物生活的环境
  • 承诺对宠物负责，不离不弃

领养流程

  1. 浏览可领养宠物列表，选择心仪的宠物
  2. 填写领养申请表，提交个人信息和领养原因
  3. 等待审核，工作人员会与您联系
  4. 审核通过后，签署领养协议，接走宠物

领养后须知

  • 定期带宠物接种疫苗，进行健康检查
  • 为宠物提供均衡的饮食和适当的运动
  • 对宠物进行必要的训练和社交化
  • 如遇特殊情况无法继续饲养，请联系我们

🌟 我们的服务

1. 宠物健康检查

基础体检：
  • 体温、心率、呼吸频率测量，评估生命体征是否稳定。
  • 体重与体况评分（BCS），判断是否存在肥胖或消瘦问题。
  • 皮肤与被毛检查，排查皮肤病、寄生虫（如跳蚤、螨虫）。
深度筛查：
  • 血常规与生化检查，评估肝肾功能、血糖、电解质水平。
  • 粪便检查，排查肠道寄生虫（如蛔虫、球虫）。
  • 传染病筛查（犬瘟、细小、猫瘟、猫艾滋/白血病等）。
  • 耳道、口腔、眼部检查，及时发现炎症或损伤。

2. 疫苗接种

犬类疫苗：
  • 核心疫苗：犬瘟热、细小病毒、腺病毒（二联/四联/六联疫苗），首免需完成3针，之后每年加强1针。
  • 推荐疫苗：狂犬病疫苗（法律强制接种），每年加强1针；可选择接种钩端螺旋体、犬副流感等疫苗。
猫类疫苗：
  • 核心疫苗：猫瘟、猫杯状病毒、猫疱疹病毒（三联疫苗），首免需完成2针，之后每年加强1针。
  • 推荐疫苗：狂犬病疫苗（部分地区强制），每年加强1针；可选择接种猫白血病（FeLV）疫苗（针对户外猫）。
接种须知：
  • 接种前需确认宠物健康，无发热、呕吐、腹泻等症状。
  • 接种后留观30分钟，避免过敏反应；一周内避免洗澡与剧烈运动。

3. 绝育服务

适用对象：
  • 犬猫满6个月（性成熟前），无繁殖计划的宠物。
  • 存在行为问题（如发情期乱叫、乱尿、攻击性）的宠物。
手术内容：
  • 公犬/公猫：去势术（摘除睾丸），手术时间短，恢复快。
  • 母犬/母猫：卵巢子宫摘除术（绝育术），可彻底避免发情、子宫蓄脓、乳腺肿瘤等疾病。
术后护理：
  • 术后佩戴伊丽莎白圈，防止舔舐伤口。
  • 7~10天内限制剧烈运动，保持伤口干燥清洁。
  • 遵医嘱服用止痛药与抗生素，定期复查拆线。

4. 领养咨询

匹配评估：
  • 根据领养人的居住环境（如是否有阳台、是否合租）、作息时间、经济能力，推荐适合的宠物体型与品种。
  • 针对有小孩/其他宠物的家庭，提供兼容性评估与相处指导。
知识科普：
  • 讲解不同品种的性格特点、饲养难度、日常开销（如粮食、医疗、美容）。
  • 科普宠物应激反应、分离焦虑等常见问题的应对方法。
后续支持：
  • 提供领养后1个月内免费咨询服务，解答喂养、训练、健康等问题。
  • 建立领养人社群，方便经验交流与互助。

📋 服务流程

  1. 咨询：

    线上/线下咨询领养要求、费用、宠物信息，了解领养协议与责任。工作人员会了解您的生活习惯、居住条件，初步匹配合适的宠物。

  2. 选择：

    浏览待领养宠物档案（包含年龄、品种、性格、健康状况、照片/视频）。预约线下见面，与心仪宠物互动，观察相处状态。

  3. 申请：

    填写《领养申请表》，提交身份信息、居住证明、联系方式。同意并签署《领养协议》，明确双方权利与义务（如不得遗弃、定期回访）。

  4. 审核：

    工作人员进行背景审核（电话/上门家访），确认领养人具备饲养条件。审核通过后，通知领养人办理交接手续。

  5. 交接：

    领取宠物及健康档案（疫苗本、驱虫记录、绝育证明）。工作人员现场讲解喂养注意事项、应急处理方法。缴纳少量领养押金（部分机构设置，满一定期限后退还）。

  6. 回访：

    领养后1周、1个月、3个月进行线上/线下回访，跟踪宠物适应情况。提供持续的饲养指导与医疗建议，确保宠物幸福生活。

💡 温馨提示

  • 领养前请慎重考虑，宠物是十几年的陪伴，需承担长期的经济与时间责任。
  • 拒绝“冲动领养”，确保您和家人都做好了迎接新成员的准备。
  • 领养代替购买，给流浪动物一个温暖的家。

🐾 宠物小知识

1. 如何正确喂养狗狗

营养均衡：

  • 蛋白质：占日粮20%~30%，来源为优质肉类（鸡胸肉、牛肉）、蛋类、豆制品，用于肌肉修复与生长。
  • 脂肪：占日粮10%~15%，提供能量并维护皮肤健康，避免过量导致肥胖。
  • 碳水化合物：占日粮50%~60%，以糙米、燕麦等粗粮为主，提供持久能量。
  • 维生素与矿物质：通过蔬菜（胡萝卜、西兰花）、专用宠物保健品补充，维持骨骼与免疫功能。

分龄喂养：

  • 幼犬（2~12月）：少食多餐（3~4次/天），选择高蛋白幼犬粮，避免喂食牛奶（易乳糖不耐受）。
  • 成犬（1~7岁）：每日1~2餐，根据体型（小型/中型/大型）控制食量，避免暴饮暴食。
  • 老年犬（7岁+）：选择低脂肪、高纤维的老年犬粮，补充关节养护类营养（如葡萄糖胺）。

禁忌食物：

严禁喂食巧克力、葡萄/葡萄干、洋葱、大蒜、木糖醇、酒精类食物，这些会导致中毒甚至死亡。

2. 狗狗的日常护理

毛发护理：

  • 短毛犬：每周用橡胶梳梳理1次，去除浮毛；长毛犬（如金毛、萨摩耶）需每日梳理，防止打结。
  • 洗澡：夏季每1~2周1次，冬季每2~3周1次，使用宠物专用沐浴露，避免频繁洗澡破坏皮肤油脂层。

清洁与健康检查：

  • 指甲：每2~3周修剪1次，避开血线，避免剪伤出血；若狗狗抗拒，可使用磨甲器。
  • 耳朵：每周用宠物洗耳液清洁1次，观察是否有异味、红肿，预防耳螨与中耳炎。
  • 眼睛：每日用生理盐水擦拭眼角分泌物，若出现持续流泪、红肿，需及时就医。
  • 口腔：每周刷牙2~3次，使用宠物牙膏与牙刷，预防牙结石与牙周病。

1. 如何正确喂养猫咪

营养核心：

  • 蛋白质：猫咪是纯肉食动物，日粮中蛋白质占比需≥30%，优先选择动物蛋白（鸡肉、鱼肉、牛肉）。
  • 脂肪：占比15%~20%，提供能量并维护毛发光泽；避免高碳水食物（如过量谷物），易导致肥胖与糖尿病。
  • 牛磺酸：必须额外补充，缺乏会导致视网膜退化与心脏疾病，可通过专用猫粮或宠物保健品获取。

分龄喂养：

  • 幼猫（2~12月）：少食多餐（3~4次/天），选择幼猫奶糕或高蛋白幼猫粮，可搭配羊奶粉补充营养。
  • 成猫（1~7岁）：每日2餐，可采用干湿粮结合喂养，保证每日充足饮水（避免长期只喂干猫粮）。
  • 老年猫（7岁+）：选择低磷、易消化的老年猫粮，补充益生菌维护肠道健康。

喂养禁忌：

严禁喂食洋葱、大蒜、巧克力、葡萄、百合科植物（如百合、郁金香），这些对猫咪有剧毒。

2. 猫咪的日常护理

毛发护理：

  • 短毛猫：每周用硅胶梳梳理1次，去除浮毛；长毛猫（如布偶、波斯猫）需每日梳理，重点梳理腋下、腹部等易打结部位。
  • 换毛季（春秋）：增加梳理频率，可使用祛毛梳，减少毛球症风险。

生活环境与健康：

  • 猫砂盆：每只猫咪至少配备1个猫砂盆，每日清理粪便，每周彻底更换猫砂并清洗盆具，避免细菌滋生。
  • 活动空间：提供猫爬架、猫抓板，满足攀爬与磨爪需求；放置阳光充足的窗台，让猫咪享受日光浴。

健康检查：

  • 耳朵：每周观察是否有黑褐色分泌物、异味，若有需排查耳螨。
  • 眼睛：每日擦拭分泌物，若出现结膜充血、畏光，需及时就医。
  • 口腔：每周检查牙龈是否红肿，定期刷牙或使用洁牙零食，预防口炎。
